# TeamPro 線上回報後端 — 部署說明

讓「選手用自己的手機回報」「教練後台看誰沒回報」「家長查閱」可以跨裝置運作。
後端用 Google Apps Script + Google Sheet（免費、用你自己的 Google 帳號）。

## 一、部署 GAS（只做一次，約 3 分鐘）

1. 開 https://script.google.com → 「新增專案」
2. 把 `Code.gs` 的全部內容貼進去，存檔（Ctrl+S），專案命名例如 `TeamPro 後端`
3. 右上「部署」→「新增部署作業」
   - 齒輪選「**網頁應用程式**」
   - 說明：隨意（例如 v1）
   - 執行身分：**我（你的帳號）**
   - 具有存取權的使用者：**任何人**
4. 按「部署」，第一次會要求授權 → 允許
5. 複製結尾是 **`/exec`** 的網址（例如 `https://script.google.com/macros/s/AKfy.../exec`）

> 資料表（Roster、Reports）會在第一次有人使用時自動建立，不用手動開。

## 二、在 TeamPro 啟用

1. 開 TeamPro → 底部「設定」→ 找到「🌐 線上回報」
2. 把剛剛的 `/exec` 網址貼上 → 按「儲存後端網址」
3. 先到「選手名單」把選手加進去
4. 按「**啟用並產生分享連結**」→ 系統會把名單上傳，並給你兩個連結：
   - **選手回報連結**：丟到隊伍 LINE 群，選手用自己手機開、選名字、填回報
   - **家長查閱連結**：給家長開、選孩子、看最近狀態（唯讀）

## 三、教練看誰沒回報

設定或戰情室裡按「**查看線上回報狀況**」→ 會即時從雲端列出：
- 今日已回報名單
- 今日尚未回報名單
- 各選手今日燈號

## 常見問題

- **改了 Code.gs 之後要重新部署嗎？** 要。部署 →「管理部署作業」→ 編輯（鉛筆）→ 版本選「新版本」→ 部署。網址不變。
- **連結打開沒資料？** 確認設定裡的 `/exec` 網址正確、且部署權限是「任何人」。
- **想換一份 Sheet？** 把 `Code.gs` 最上面的 `SHEET_ID` 填成你的 Google Sheet ID 即可。
